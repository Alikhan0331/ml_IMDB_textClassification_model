# IMDB Movie Review Sentiment Classification

## Overview
This project builds a machine learning model to classify **IMDB movie reviews** as **positive** or **negative** based on the text of the review. It walks through a typical NLP workflow: loading text data, preprocessing and vectorizing it, training a classifier, and evaluating the model on unseen reviews.

## Dataset
- Each sample:
  - **input**: review text (free‑form English text),
  - **label**: sentiment (`positive` / `negative`).

## Project Goals
- Preprocess raw movie reviews so they can be fed into a machine learning model.
- Train a baseline text classifier to predict sentiment.
- Evaluate performance using standard classification metrics.
- Demonstrate how to use the trained model to predict sentiment for new, custom reviews.
